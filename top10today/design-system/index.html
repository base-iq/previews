Usage rules

  • The logo is ink. Ink on light, white on ink. The solid slate avatar is the single color exception.
  • Clear space: the height of the wordmark's capitals on all sides.
  • Minimum sizes: flag 140px wide; compact flag 90px; avatar 24px.
  • The rules are part of the mark. Never set the wordmark without them, never thicken them past hairline weight (1px at text sizes, 2px at avatar sizes).
  • Don't add color to the wordmark, letterspace beyond the specified .22em, set it in PT Serif, or place it on photography without a paper-colored field.
  • Files (live, produced 2026-07-16): top10today-flag-{light,dark}.svg/.png, top10today-avatar-{outline,slate,ink}.svg/.png — SVG as outlined paths (no font dependency), PNG at large raster sizes, at previews/top10today/assets/logos/. The flag scales; there is no separate compact file — switch to the avatar below ~90px. The previous ring-based set is removed (git history retains it).

03 — Type

Typography: the PT pair

PT Serif for headlines and rank numerals; PT Sans for everything else. The two faces share skeletons — commissioned as a pair for maximum public legibility. Nothing tracks except the wordmark and small caps furniture; nothing exclaims.

Scale

Body text carries the blurbs at 17px with generous leading — the audience is 50+ and reading on phones. It never drops below the floor.

Intro stamp · PT Sans 700 11.5px +0.18em caps

Fine print · PT Sans 400 13.5px

Items marked * are paid placements. The countdown is editorial.

Heading style: sentence case everywhere; all-caps lives only in the wordmark, date row, and intro stamps. Rank numerals are always PT Serif 700 with a period ("7."). Numerals in running text are PT Sans.

Email typography — designed on the fallback

Gmail and Outlook don't render web fonts; only Apple Mail does. The email stack is designed on faces every device ships: Georgia inherits PT Serif's role, Helvetica/Arial inherits PT Sans. Web faces listed first as progressive enhancement. Because the system has no rings, badges, or devices, nothing else degrades — the numeral column is live text in every client.

04 — The unit

The ranked item: number, headline, blurb

The most-repeated component in the brand. Every ranked list counts down to #1 — a three-item section runs 3-2-1; the flat ten runs 10→1. Numerals sit right-aligned in a fixed column so the eye can run straight down the ranks.

The finale — #1, always last

The Riverside Market reopens Saturday, two years after the flood

44 stalls, a covered arcade, and the same peach lady in slot #3. Gates open at 8 a.m.; parking fills by 9. Worth the early alarm.

The whole finale treatment: intro stamp, the numeral in slate at 34px, headline up to 21.5px. Emphasis by weight and size — no filled devices, no motion. One finale per page or email; in a multi-section edition, the lead section's #1 takes it and other sections end at a standard row.

Compact — recaps, secondary lists

10.Little League signups close Friday
9.The heat advisory holds through Thursday
8.Night markets return to Dock Street

Rules of the countdown

  • Every list descends. Rank 1 is always the last item rendered. No exceptions.
  • One finale per surface, marked by the stamp + slate numeral + larger headline. Nothing else in a list is slate.
  • Blurbs are 1–3 sentences. If it needs more, it's an article — link out.
  • Placements are ranked items. A paid item occupies a rank position; there is no "between ranks."
05 — Interactive

Buttons, links, forms

Quiet 10px corners — square enough to feel like print, soft enough to feel tappable. One primary button per viewport. Focus is a 2px slate outline on every interactive element.

Buttons — Primary (slate) · Secondary (outline) · states

PT Sans 700, sentence case, never all-caps. Hover deepens color only — no movement, no shadows anywhere in the system except genuinely floating overlays (0 2px 12px rgba(32,38,44,.16) on sticky bars and dropdowns only).

Link registers

Form fields

One email each morning. Unsubscribe anytime.

⚠ That address is missing its ending — try something like name@example.com.

White fields, 1px Rule border, 10px radius, 46px min-height, PT Sans 16px. Error copy always says the fix. Full field matrix (select, textarea, checkbox, radio) carries over from the previous system with these tokens.

Motion — almost none

Color-only transitions at 120ms on hover and focus. No entrance animations, no scroll effects, no pulses. prefers-reduced-motion collapses even the color transitions. The calm is the brand.
