Top10Today — The Bulletin, logo variations · Round 5 · July 16, 2026

Five logos that don't perform

All monochrome, all typographic, all in The Bulletin's two faces. Each is shown four ways: primary lockup, reversed on ink, in the masthead with real content under it, and reduced to avatar/favicon sizes — the small sizes are where wordmark-only systems usually fail, so every option here has a designed reduction, not an afterthought.

Rules shared by all five

  • Strictly monochrome. Ink on light, white on ink. No third color exists in the logo system.
  • Two faces only. Inter and Merriweather — the same faces as the product. (If variation P's Source Serif wins round 4, it swaps in cleanly.)
  • No pictorial device. No rings, arches, seals, or tallies. The reductions use the numeral because a favicon has to say something in 32 pixels — not because the brand worships it.
  • Production plan (proposal stage): chosen logo ships as SVG (outlined paths) + PNG 2x in light/dark variants, same file convention as the current asset set. Existing logo files stay in place until the new set is approved.

Composing: the logo choice is independent of round 4's type/density/color choices — any logo here works with any Bulletin variation. Pick a number (or "3 but with the serif numeral," etc.) and the consolidated proposal gets built from the full stack of picks: Bulletin base + round-4 details + this logo.
