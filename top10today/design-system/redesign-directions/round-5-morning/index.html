The connection: in The Morning's product, the #1 numeral is the only slate numeral in the list — and it's this exact serif. Masthead, favicon, and finale all repeat one small idea. Note: slate at logo sizes is 6.9:1 on this paper — safe at every size shown.

Rules shared by all five

  • Two faces only. PT Serif and PT Sans — the same faces as The Morning's product.
  • Slate is budgeted, not default. Options 1–2 keep the logo all-ink (slate only in solid avatars); options 3–5 each use slate in exactly one stated place. Nothing is blue "because it looks nice."
  • No pictorial device. Same rule as the Bulletin board — reductions use the numeral because 32 pixels demands it.
  • Production plan (proposal stage): SVG (outlined paths) + PNG 2x, light/dark variants, current file convention. Existing logo files untouched until approval.

Comparing the two directions: this board mirrors The Bulletin's logo board treatment-for-treatment — open them side by side and the difference you're weighing is exactly two things: Merriweather/Inter's newsprint sturdiness vs. PT's rounder warmth, and zero color vs. one careful blue.
