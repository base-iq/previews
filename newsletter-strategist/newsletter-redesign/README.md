# BaseLocal Newsletter — Tightened v2

Self-contained HTML newsletter ready to deploy to GitHub Pages or any static host.

## Files
- `index.html` — the newsletter
- `styles/newsletter.css` — base styles (BaseLocal tokens, layout)
- `styles/newsletter-v2.css` — tightening overrides (kicker labels, masthead, hairlines, advertise CTA, top-nav polish)
- `assets/glendale-cover.jpg` — masthead banner
- `assets/baselocal-logo-cream.webp` — footer logo

## Deploying to GitHub Pages
1. Push this folder to a repo (or to a subfolder of one).
2. In repo settings → Pages → set the branch and path.
3. Visit the published URL — `index.html` is the entry point.

The Caveat font is loaded via Google Fonts; everything else uses system stacks (Bitter / Lato fall back to Georgia / Helvetica).
